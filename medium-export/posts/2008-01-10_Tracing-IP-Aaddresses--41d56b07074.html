Tracing IP Aaddresses

OK, here is a new thing I recently learned about.

Do you know that you can open up other people’s router pages ?

Mind you, this is not exactly legal (Even my telling you this is not legal).

So having told you that, let’s begin.

First of all, you need to know your own IP address.

Try this link : http://www.ip-adress.com/

Now that you know your IP address,download this free program,

called Free IP Scanner (Yeah,that is what it is called).

It is a free download. Link :http://www.eusing.com/ipscan/free_ip_scanner.htm

Now run the program and type in your IP address in the box. Instead of the last

3 digits (after the 3rd dot) type in 1 and in the other box type in the same thing

accept this time enter the last digits (Yes, after the 3rd dot) as 255. Now press

scan.

A whole list will pop up with numbers in the left hand corner. Some will have a
tick before them, others will have a cross. Now choose any number with a tick.
Enter this number into the address bar of your browser and press enter. A
dialog box opens up asking for username and password. Now the guess work
begins. Most of the times, username is admin and password is also admin or
password. Try out others as well. And yeah, it is ILLEGAL. I have tried it
once or twice and it has worked. Also, change the password of your router
because the same thing can happen to you.

And one more thing, like ethical hackers, please do not change the settings of
the router who’s homepage you’ve opened. Or else you’ll be in serious trouble
