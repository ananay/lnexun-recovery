Turbo C++ has been a part of CBSE’s Class 11th and 12th Computer Science curriculum for a long time now. Being a Class 12th Computer Science student myself, I’ve seen almost everyone struggling to install Turbo C++ on their computers. Although, the people using Windows have been able to get it done somehow, but the people who have been using Mac for a while haven’t been able to get it to work.

One way to get this to work was using DOSBox for Mac, which I do. But of course, the installation process ain’t that user friendly. My friend, Shreyas Kishore decided to make the installation simpler by packing it into a simple DMG file. So here’s how you go about it.

First off, Download Turbo C++ for Mac by using one of these links:

http://exun.co/turboc

http://ananayarora.com/turbo.dmg

The installation process from this point is pretty easy, but if you still don’t know how to go about it, follow the rest of this article.

Open the file, and wait for this to happen.

Then just drag and drop, the turbo C++ folder into the ‘Applications’ folder.

It should now be in the ‘Applications’ folder on your mac.

Open up the folder, and now you’ll have this screen. But wait, When you’re running it for the first time, instead of double clicking it, Right click and hit ‘Open’ as shown. This is because Macs have special security which you need to pass through, and that’s how you go through it.

Annddd you’re good to go.
