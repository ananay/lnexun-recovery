Fixing Facebook’s Chat Box

Recently, Facebook’s been trying to make it’s users switch over to Facebook Messenger on every platform. While they do this, they changed their Facebook’s chat box scheme to messenger-styled boxes.

As soon as this color changed, it caught my eye. It was literally unbearable to see the white top bar along with the white news feed making the chat boxes really weird. 

Being a developer, I thought I could fix this. I thought of writing a chrome extension so as to get around this. My friend who was in the US and me being in India told me that the change wasn’t there on his computer. So I changed my time to San Francisco’s Time, fired up my VPN, logged on to my secondary account and got the old chat boxes. I copied the CSS from there and pasted it into a new file. 

Chrome has a very simple way of injecting CSS / Javascript through extensions. I found it very easy as I just had to create a manifest.json file in order to inject the CSS into Facebook.

It was a quick 5 minute task that fixed that disturbing chat box color. I’ll be soon putting this on the Chrome Web Store once I buy a license for that. Until then you can download the extension via the link below.

Here’s the extension if you guys need it : http://ananayarora.com/facebook_chat_box.crx
