Installing the new version of Turbo C++ [MacOS]

Yes, I know I’ve been procrastinating a lot but it’s finally here. In fact it was ready for MacOS before the Windows version but I was too lazy to package it for downloading. Anyway, let’s get right into it.

If you’re looking for the Windows version — Click Here

The new version of Turbo C++ I’ve created is essentially a combination of an editor called “Sublime Text” and the Turbo C++ compiler itself; so here’s how to install it.

Download the Mac version from here: https://ananay.co/turbocppmac

Open it up, and you should be presented with this:

Drag and drop the “TurboC++Mac.app” file into the “Applications” folder as shown. This step is important or it will fail to run.

Click on “Applications” option on the side and you should see “TurboC++Mac.app” in the Applications list. Double click to open it.

Double click on it and you should get this:

To get rid of this message, right click and hit open.

Once you do that, you should get an option to open it. Hit open.

You should get this on your screen:

Next, go to Tools > Build System and make sure “turbo” is selected as shown.

This is an important step or your code won’t be compiled.

Sweet! Everything is setup now.

Using the editor

Go to File > New File (or using Command + N). Without typing any code, save the file first by doing File > Save (or using Command + S).

Next, write your code into the code window. I’m just going to make a quick Hello World program for this demo.

Save your code using File > Save (or Command+S) and then run your code using Tools > Build (or using Command + B). Once you do that you should see this:

*drumroll* Press any key to continue

And that’s all folks! Here’s all you have to do to get it up and running on your Mac. Should you run into any problems with this, feel free to send me an email, shoot me a message on Facebook or on Twitter.
